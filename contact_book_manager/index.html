Project 2 (Password Strength Validator):

The **Password Strength Validator** is a Python-based utility designed to assess the robustness of user passwords based on a set of predefined security rules. It checks whether a password meets key criteria such as a minimum length of 8 characters, the inclusion of at least one uppercase letter, one lowercase letter, one digit, and one special character (like `!@#$%^&*()-_+=`). Based on how many of these conditions are satisfied, the validator classifies the password as **Strong**, **Moderate**, or **Weak**, thereby encouraging users to create more secure credentials. This tool is especially useful for applications that require basic client-side validation before password storage or submission, helping reduce vulnerability to attacks like brute force or credential stuffing.
